# A Look at Civic Life — Project Plan

Use this file for the project timetable, goals and launch route.

This project uses the Project Web Builder method: task board, current task, project plan, decision log and not-yet list.

## Project goal

Create a clear, calm, public-facing civic field journal for **A Look at Civic Life — 52 Weeks for the Common Good**, ready for a quiet Week 0 launch and then a four-week pilot.

## Launch-ready definition

The project is launch-ready when:

- the public site works on mobile and desktop;
- Home, About, 52 Weeks, Current Week, Campaign Log, Method and Contact are coherent;
- Privacy, Cookies, Accessibility and Terms have been reviewed;
- the project email/contact route is confirmed;
- the Contact page email button is live;
- the Campaign Log is empty except for real public civic actions;
- Week 0 is ready for quiet launch;
- SEO and sharing basics are in place;
- no blog, polls, comments or mailing list have been added during the pilot;
- Week 1 is ready to begin as the first real campaign week.
