# Codex First Task

Build the website skeleton only. Do not write full campaign content yet.

## Project

**A Look at Civic Life — 52 Weeks for the Common Good**

## Build requirements

Create a static website using plain HTML, CSS and JavaScript.

Do not use React unless genuinely necessary.

Do not use a complex build system.

The site must be easy for Alan to edit manually.

## Required pages

- `index.html`
- `about.html`
- `52-weeks.html`
- `current-week.html`
- `campaign-log.html`
- `method.html`
- `contact.html`
- `privacy.html`

## Required data files

- `data/campaigns.json`
- `data/log.json`

## Design direction

Calm British/Irish civic tone. Use neutral colours. Avoid card soup, Silicon Valley design, partisan colours, fake campaign gloss, AI hype and shouting.

Think: public information project; civic field journal; evidence log; readable public document; thoughtful, warm and dignified.

## Required functionality

- 52 Weeks tracker should load from `campaigns.json`.
- Campaign Log should load from `log.json`.
- Current Week page should be reusable.
- Site should be mobile responsive.
- README should explain how Alan updates the site weekly.

## Acceptance criteria

The first build is successful if all required pages and data files exist, the tracker/log work, the visual design is calm and readable, and Alan can understand which files to edit each week.
