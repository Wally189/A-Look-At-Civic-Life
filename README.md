# A Look at Civic Life — Project Control v0.2

This repository is the working home for **A Look at Civic Life — 52 Weeks for the Common Good**.

The project-control system is designed for a simple working rhythm:

1. Open `project-control/TASK_BOARD.md`.
2. Find the first task marked `Next` or `In progress`.
3. Open the matching task card in `project-control/task-cards/`.
4. Work through the checkboxes.
5. Mark the task as `Done`.
6. Mark the next task as `Next`.

For ChatGPT, say:

> Use the task board. What task are we working on?

Then say:

> Break this task into mini tasks.

The task board is also saved as:

- `project-control/data/task-board.json`
- `project-control/data/task-board.csv`

The JSON version can later drive a simple website/project dashboard.

Core sentence:

> Make public life readable. Then make it answerable. Then make it fair.
